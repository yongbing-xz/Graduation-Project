# 硬件兼容性检测平台 - 注册登录功能使用说明

## 功能概述
已成功为硬件兼容性检测平台添加了完整的注册登录系统，包括：

- ✅ **用户注册**：新用户可以注册账号
- ✅ **用户登录**：现有用户可以登录系统
- ✅ **登录状态检查**：自动跳转到登录页面
- ✅ **用户信息显示**：登录后显示欢迎信息
- ✅ **退出登录**：用户可以安全退出
- ✅ **自动跳转**：登录成功后自动跳转到主页面

## 使用流程

### 1. 新用户注册
1. 打开 `login.html` 页面
2. 点击"立即注册"链接
3. 填写用户名、邮箱、密码和确认密码
4. 点击"注册"按钮
5. 注册成功后自动跳转到登录页面

### 2. 用户登录
1. 在登录页面填写用户名和密码
2. 点击"登录"按钮
3. 登录成功后自动跳转到主页面 `未命名.html`

### 3. 使用平台
- 登录后可以在主页面右上角看到欢迎信息
- 可以正常使用硬件兼容性检测功能
- 点击"退出登录"可以安全退出系统

## 技术实现

### 数据存储
- 使用浏览器的 `localStorage` 存储用户数据
- 用户信息包含：用户名、邮箱、注册时间等
- 登录状态持久化存储

### 安全特性
- 密码长度验证（至少6位）
- 邮箱格式验证
- 确认密码一致性检查
- 用户唯一性检查（用户名和邮箱不能重复）

### 用户体验
- 响应式设计，适配各种屏幕尺寸
- 加载状态提示
- 错误信息友好提示
- 成功操作的视觉反馈

## 文件说明

- `login.html` - 登录注册页面
- `未命名.html` - 主页面（已添加登录检查功能）
- 用户数据存储在浏览器本地

## 测试方法

1. 直接打开 `login.html` 测试注册功能
2. 注册新用户后测试登录功能
3. 登录后验证自动跳转和用户信息显示
4. 测试退出登录功能
