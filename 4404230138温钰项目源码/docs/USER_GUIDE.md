# 硬件兼容性检测平台 - 用户指南

## 概述

硬件兼容性检测平台是一个专业的网页工具，帮助用户验证电脑硬件组件的兼容性。支持CPU、主板、显卡、内存、硬盘、机箱等主要组件的兼容性检测。

## 快速开始

### 1. 启动应用

- 双击运行 `启动.bat` 文件
- 或直接打开 `index.html` 文件
- 使用默认账号登录：用户名 `demo`，密码 `123456`

### 2. 基本使用流程

1. **登录系统**
   - 首次使用需要注册账号或使用默认账号
   - 登录后自动跳转到主界面

2. **选择组件**
   - 点击左侧组件卡片上的"选择"按钮
   - 在弹出的选择器中选择具体型号
   - 支持按品牌、关键词搜索筛选

3. **查看兼容性结果**
   - 右侧区域实时显示已选组件的参数
   - 兼容性检测结果实时更新
   - 红色表示关键问题，黄色表示警告，绿色表示正常

## 功能详解

### 智能筛选功能

- **开启智能筛选**：勾选后，系统会自动过滤与已选组件不兼容的选项
- **自动清理**：选择新组件时，自动清理不兼容的已选组件

### 数据管理

- **导入数据集**：支持从本地文件或远程URL导入硬件数据
- **导出配置**：将当前配置导出为JSON格式
- **数据验证**：自动验证导入数据的完整性和有效性

### 兼容性检测规则

系统内置以下主要检测规则：

1. **CPU与主板插槽匹配**
2. **内存与主板DDR代数匹配**
3. **机箱与主板尺寸兼容**
4. **显卡与机箱长度兼容**
5. **CPU散热器与机箱高度兼容**
6. **电源功率建议**

## 高级功能

### 关键词生成

当搜索结果为空时，可以：
- 点击"生成关键词"按钮
- 获取针对性的搜索建议
- 复制关键词到豆包网站查询
- 导入查询结果到本地数据集

### 数据验证报告

- 自动验证组件数据的完整性
- 生成数据质量报告
- 提供数据改进建议

## 常见问题

### Q: 如何添加新的硬件型号？
A: 使用"导入数据集"功能，支持JSON格式的数据导入。

### Q: 兼容性检测的准确度如何？
A: 系统基于公开的硬件规格信息进行检测，准确度取决于数据完整性。

### Q: 支持移动设备吗？
A: 支持，系统采用响应式设计，适配各种屏幕尺寸。

### Q: 数据会保存在哪里？
A: 数据保存在浏览器本地存储中，不会上传到服务器。
