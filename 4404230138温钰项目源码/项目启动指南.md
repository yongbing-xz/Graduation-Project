## 🚨 常见问题解决

### ❌ 端口占用问题
**症状**: 端口8080或5173被占用
```bash
# 查看端口占用
netstat -an | findstr ":8080"
netstat -an | findstr ":5173"

# 解决方案1: 关闭占用进程
taskkill /PID <进程ID> /F

# 解决方案2: 更改端口
# 修改 application.yml 中的 server.port
```

### ❌ Maven依赖下载慢
**解决方案**:
```bash
# 使用国内镜像
# 编辑 settings.xml 添加阿里云镜像
<mirror>
    <id>aliyunmaven</id>
    <mirrorOf>*</mirrorOf>
    <name>阿里云公共仓库</name>
    <url>https://maven.aliyun.com/repository/public</url>
</mirror>
```

### ❌ Node.js依赖安装失败
**解决方案**:
```bash
# 清理缓存
npm cache clean --force

# 使用国内镜像
npm config set registry https://registry.npmmirror.com

# 重新安装
rm -rf node_modules package-lock.json
npm install
```

### ❌ 前端启动失败
**可能原因**:
- Node.js版本过低
- 端口5173被占用
- 依赖安装不完整

**解决方案**:
```bash
# 检查Node.js版本
node -version

# 重新安装依赖
rm -rf node_modules
npm install

# 使用不同端口
npm run dev -- --port 3000
```
